# Apartment Booking System

## Overview
This is a simple command-line Python project for booking various types of apartments (2BHK, 3BHK, 4BHK, Penthouses) with features such as pricing logic for corner apartments, token-based booking, and conditional billing based on payment mode. The system demonstrates basic procedural programming, user interaction, and receipt generation for a real-estate scenario.

## Features
- Supports booking for 2BHK, 3BHK, 4BHK, and Penthouse units
- Special price calculation for corner apartments
- Early registered customers enjoy club benefits
- Accepts multiple payment methods (cheque, cash, card, UPI)
- Generates bill receipts with status and a unique token number
- Handles cheque-specific billing conditions

## Technologies / Tools Used
- Python 3.x
- Standard Python libraries (no external dependencies)
- Runs via command line / terminal

## Instructions for Testing
- Follow the on-screen prompts to enter your details and select an apartment type
- Specify if you need a corner apartment
- Choose your payment method (test both 'cheque' and other methods for receipt variations)
- Verify the generated receipt, bill amount, club eligibility, and payment status on screen
